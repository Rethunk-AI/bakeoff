## Settled
- `schema_version`: simple integer, present from day one.
- Disk layout: directory-per-table, UUID-filename records under `BAKEOFF_DATA_DIR` (`run_queue` is the ephemeral exception).
- Ingestion: URL-submit → metadata-pull → validate → store. No hand-written JSON except admin/seed; no CI generation; embedded-GGUF-metadata cross-check deferred to P2.
- Embedding map: `creator` + `sources` embedded in the model record; metrics embedded in the run record.
- `created_at` / `updated_at` = disk-record-lifecycle audit fields.
